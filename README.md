# Hi, I'm @zenhouke

I build **human-in-control AI agents** and dependable infrastructure automation.

My work sits at the intersection of agent systems, operational tooling, and long-lived memory. I care about software that makes powerful automation observable, explainable, and safe to operate.

## What I'm working on

- **[Ops Agent](https://github.com/zenhouke/ops-agent)** — an AI-assisted operations console where agents can plan and explain, while operators retain final control over command execution.
- **[MemFlow](https://github.com/zenhouke/MemFlow)** — a Go SDK for lifelong agent memory with automated tiering and semantic compression.
- Infrastructure tools for monitoring, OpenWrt, knowledge systems, database backup, and repeatable operations.

## How I build

- Keep humans at the approval boundary for consequential actions.
- Make plans, tool calls, command output, and failures observable.
- Prefer small, dependable systems over opaque automation.
- Design agent memory for long-running work, not just single conversations.
